Deployment settings

The generated JSON is safe to commit. Set operational values in GitHub Settings > Secrets and variables > Actions or Cloudflare Worker > Settings > Variables and Secrets only.

GitHub Actions secrets
  • ANTHROPIC_API_KEY for Claude search, ranking, and summaries.
  • SMTP_HOST, SMTP_PORT, SMTP_USER, SMTP_PASS for email delivery.
  • FROM_EMAIL and ADMIN_EMAIL for sender identity and Sunday review.
GitHub Actions variables
  • RESEARCH_DOMAIN_PROFILE set to <profile-id>.json.
  • SITE_URL set to the public GitHub Pages URL.
  • Optional tuning: EMAIL_MAX_PAPERS, DISCOVERY_WINDOW_DAYS, CLAUDE_SEARCH_MODEL, CLAUDE_FAST_MODEL.
Cloudflare Worker secrets
  • GITHUB_TOKEN: fine-grained token for the private repo with Contents read/write and Metadata read-only.
  • ADMIN_TOKEN: long random token used by the admin page.
Cloudflare Worker variables
  • GITHUB_OWNER, GITHUB_REPO, GITHUB_BRANCH, ALLOWED_ORIGIN, and DEFAULT_CATEGORY.
  • After deployment, put the Worker URL into this builder and into the public site config.

Output file

Save as config/domains/<profile-id>.json in a fresh Paper Express clone. Never paste keys, tokens, passwords, subscribers, ratings, or digest archives into the profile.
